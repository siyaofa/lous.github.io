## 模式识别

    个人片面的理解：机器学习和深度神经网络本质上都是分类器。

    只不过有些分类器的模型复杂度很高，如果样本量足够大。模型关键系数被确定下来了，便可以有很好的分类效果。

    随着理论的完善，可能未来人类要做的事情就是建立模型和确定如何提取特征的方法了。

    目前能产生较大经济效益的主要集中在：推荐系统、图像识别和自然语音处理。
