## TCP/IP起源

早期计算机只能用于本地计算，当想和其他计算机交换信息时，过程十分复杂。

为解决计算机间通讯问题，人们提出了一套标准化协议，即TCP/IP协议簇。

TCP/IP并不是一个单一的协议，而是由多个不同层次的协议所组成的。

按层次划分，一般把TCP/IP协议分为四层或者七层。

## OSI参考模型

OSI参考模型把TCP/IP分为七层：

```mermaid

graph LR;

    应用层-->表示层;
    表示层-->会话层;
    会话层-->传输层;
    传输层-->网络层;
    网络层-->链路层;
    链路层-->物理层;

```

| 层次 | 包含的协议 |
-|-
| 应用层 | FTP |
| 表示层 | 会话层 |
| 会话层 | 传输层 |
| 传输层 | 网络层 |
| 网络层 | 链路层 |
| 链路层 | 物理层 |
